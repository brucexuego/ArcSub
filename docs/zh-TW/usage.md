# 使用說明

## 主要流程

1. 開啟或建立專案。
2. 在 **影片獲取器** 從線上連結、本機上傳或既有專案檔準備影音素材。
3. 執行 **語音轉文字**，產生逐字稿與字幕。
4. 執行 **文字精準翻譯**，翻譯字幕內容。
5. 開啟 **影片撥放器** 搭配影片觀賞翻譯後的字幕，並調整頁面中的字幕顯示樣式。

## 選擇模型

**設定** 裡的模型小卡會依照優先順序排列。排在第一個的模型，會成為工作頁面預設顯示的模型。

**語音轉文字** 與 **文字精準翻譯** 頁面會把雲端模型和本機模型分開顯示，讓你清楚知道目前選擇的模型在哪裡執行：

- 雲端模型會向你設定的服務送出請求
- 本機模型會在模型檔安裝完成後，於你的電腦上執行

如果其中一邊沒有模型，選單會保留另一邊可用的模型，並顯示缺少模型的狀態。

## 語音轉文字建議

- 如果你已經知道音檔語言，直接選定語言通常比自動偵測更穩定。
- 需要更準確的字幕時間時，可以使用字詞時間對齊。
- 音訊有較長安靜片段，或希望先找出有說話的區段時，可以使用語音活動偵測。
- 需要依照不同說話者分組字幕時，可以使用語者分離。需要 pyannote 時，可先在 **設定** 安裝。

## 翻譯建議

- 翻譯前先檢查逐字稿，尤其是人名、品牌或專有名詞很多的內容。
- 希望同一個名詞每次都翻成一樣時，使用術語表。
- 需要指定語氣、名稱處理方式或閱讀偏好時，使用提示詞。
- 啟用字幕對齊修正，可以幫助翻譯結果維持一行來源字幕對應一行翻譯字幕；實際效果會依模型而不同。

## 本機模型

若要使用本機 ASR 或本機翻譯：

1. 開啟 **設定**。
2. 選擇本機 ASR 或本機翻譯模型區域。
3. 輸入 Hugging Face 模型 ID 並檢查。
4. 安裝模型。大型模型下載可能需要較久時間，並會以背景任務繼續執行。
5. 回到工作頁面選擇已安裝的模型。

本機模型會在你的電腦上處理內容，但需要足夠的硬碟空間與合適的硬體。雲端模型通常比較容易開始使用，但會把需要處理的文字或音訊送到你設定的服務。

## 影片撥放器

**影片撥放器** 用來搭配影片觀賞翻譯後的字幕成果。影片撥放器裡的字幕樣式設定只會影響該頁面的觀賞效果。
