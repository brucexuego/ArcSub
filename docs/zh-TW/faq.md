# 常見問題

## ArcSub 是雲端工具還是本機工具？

兩者都支援。你可以使用雲端 ASR 與雲端翻譯服務，也可以使用本機 OpenVINO 模型，或依需求混合使用。

## 可以不安裝 OpenVINO 嗎？

可以。雲端 ASR 與雲端翻譯流程可以在沒有本機 OpenVINO 模型的情況下使用。

當你希望 ASR 或翻譯在自己的電腦上執行時，才需要 OpenVINO。

## 部署時一定要安裝本機模型嗎？

不需要。本機模型可以之後再從 **設定** 安裝。這樣初始安裝比較輕量，也能讓每位使用者只安裝自己需要的模型。

## 為什麼本機模型安裝需要比較久？

本機模型可能需要下載大型檔案，並為你的電腦準備可執行的模型檔。**設定** 頁面會顯示安裝狀態。

## 為什麼 pyannote 無法使用？

pyannote 語者分離需要：

- 有效的 `HF_TOKEN`
- 已接受 Hugging Face 模型存取權限
- 已在 **設定** 完成 pyannote 安裝

同一個 `HF_TOKEN` 也可用於需要認證的私有或受限制 Hugging Face 模型。

## 我的檔案會存在哪裡？

專案檔、下載的影音素材、產生的字幕與安裝的本機模型，會存放在執行 ArcSub 的電腦上。個人 API 金鑰與本機機密應留在本機 `.env` 或 **設定** 中，不應放進原始碼版本控制。

## 影片撥放器的用途是什麼？

影片撥放器用來搭配影片觀賞翻譯後的字幕成果，並調整該頁面的字幕顯示樣式。

## 如果問題只在封裝版本中發生，該怎麼辦？

可以使用封裝資料夾中的診斷腳本，並保留產生的診斷檔供後續檢查。
