# はじめに

ArcSub は、クラウドサービスとローカルモデルの両方を使える、字幕翻訳向けの一連の作業をまとめたワークステーションです。

ArcSub では次の作業ができます。

- オンラインリンク、ローカル動画アップロード、既存プロジェクトから素材を準備する
- クラウド音声認識サービス、またはローカルモデルで音声を文字にする
- クラウド翻訳サービス、またはローカルモデルで字幕を翻訳する
- 動画プレーヤーで動画と翻訳字幕を一緒に視聴する

## 作業方法を選ぶ

音声認識や翻訳サービスの API 情報がある場合は、まずクラウド中心の流れで始められます。

自分のPC上で音声認識や翻訳を実行したい場合は、**設定** からローカル OpenVINO モデルをインストールできます。ローカルモデルは大きなダウンロード容量と、十分なPC性能が必要になる場合があります。

クラウドとローカルを組み合わせることもできます。たとえば、クラウド音声認識とローカル翻訳、またはローカル音声認識とクラウド翻訳を組み合わせられます。

## 初回設定

1. **設定** を開きます。
2. クラウド ASR モデルを追加するか、ローカル ASR モデルをインストールします。
3. クラウド翻訳モデルを追加するか、ローカル翻訳モデルをインストールします。
4. **設定** でモデルカードをドラッグして優先順を決めます。一番上のモデルが既定モデルになります。
5. pyannote 話者分離、またはアクセス承認が必要な Hugging Face モデルを使う場合だけ、`HF_TOKEN` を設定します。

## 基本の流れ

1. プロジェクトを作成、または開きます。
2. **動画取得** で素材を準備します。
3. **音声文字変換** を実行します。
4. **高精度翻訳** を実行します。
5. **動画プレーヤー** で動画と翻訳字幕を一緒に視聴します。
