# FAQ

## OpenVINO がなくても使えますか？

はい。

サービスは起動できますが、ローカルモデルのインストールは使えません。

## ローカルモデルのインストールに時間がかかるのはなぜですか？

ArcSub は Hugging Face モデルのダウンロードと変換をバックグラウンドインストールタスクとして実行します。

**Settings** に現在のタスク状態が表示されるため、大きなモデルが単一のブラウザーリクエストの長さだけで失敗扱いになるべきではありません。

## pyannote が使えないのはなぜですか？

pyannote には次が必要です。

- 有効な `HF_TOKEN`
- Hugging Face の gated model 承認
- インストール完了

同じ `HF_TOKEN` は、認証が必要な gated/private Hugging Face ローカルモデルのダウンロードにも使われます。

## データはどこに保存されますか？

通常はローカルの `runtime/` に保存されます。
