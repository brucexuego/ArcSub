# FAQ

## ArcSub はクラウドアプリですか、それともローカルアプリですか？

両方に対応しています。クラウド ASR とクラウド翻訳サービス、ローカル OpenVINO モデル、またはその組み合わせを使えます。

## OpenVINO なしで使えますか？

はい。クラウド ASR とクラウド翻訳の流れは、ローカル OpenVINO モデルをインストールしなくても利用できます。

自分のPC上で ASR や翻訳を実行したい場合に OpenVINO が必要になります。

## デプロイ時にローカルモデルを入れる必要はありますか？

必要ありません。ローカルモデルはあとから **設定** でインストールできます。初期セットアップを軽くし、必要なモデルだけを選べます。

## ローカルモデルのインストールに時間がかかるのはなぜですか？

ローカルモデルは大きなファイルをダウンロードし、自分のPCで使えるように準備する場合があります。**設定** ページでインストール状況を確認できます。

## pyannote が使えないのはなぜですか？

pyannote 話者分離には次が必要です。

- 有効な `HF_TOKEN`
- Hugging Face モデルへのアクセス承認
- **設定** で完了した pyannote インストール

同じ `HF_TOKEN` は、認証が必要な非公開または制限付き Hugging Face モデルにも使えます。

## ファイルはどこに保存されますか？

プロジェクトファイル、ダウンロードしたメディア、生成された字幕、インストール済みローカルモデルは、ArcSub を実行しているマシンに保存されます。個人の API キーやローカルの秘密情報は、ローカル `.env` または **設定** に保存し、ソース管理には入れないでください。

## 動画プレーヤーは何のためのページですか？

動画プレーヤーは、動画と翻訳字幕を一緒に視聴し、そのページ上の字幕表示スタイルを調整するためのページです。

## パッケージ版でだけ問題が起きる場合はどうすればよいですか？

パッケージフォルダに含まれる診断スクリプトを実行し、生成された診断ファイルを確認用に残してください。
